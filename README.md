# Data Analysis & Reporting- Global Evaluation Dashboard
## Project Overview
  This project presents a clean and structured analysis of CRM data using Excel.
  The dataset was carefully processed and summarized to deliver clear insights, performance metrics, and reporting dashboards suitable for academic evaluation and decision-making.
  The goal of this project is to showcase practical data analysis skills, proper spreadsheet modeling, and the ability to transform raw data into actionable insights.

## Objectives
  * Organize raw CRM data into a clean analytical structure.
  * Perform calculations related to customers' performance and scoring.
  * Build a readable and professional Excel dashboard.
  * Apply data validation, formulas, and conditional formatting for clarity.
  * Present insights in a way suitable for academic or institutional use.

## Key Features
  * Clean data structure with no duplicates.
  * Automated calculations using Excel formulas.
  * Professional formatting for clarity and readability.
  * Summary tables for:
    ** Total Leads
    ** Top Interact Countries
    ** Most Languages Needed	
  * Optional dashboard sheet for visualization.
  * Export-ready Excel file for institutions.

## Tech Stack
| Tool                        | Usage                                           |
| --------------------------- | ----------------------------------------------- |
| **Microsoft Excel**         | Data cleaning, formulas, formatting, dashboards |
| **Excel Formulas**          | SUM, AVERAGE, VLOOKUP/XLOOKUP, IF, COUNTIF      |
| **Conditional Formatting**  | Highlighting performance levels                 |
| **Pivot Tables (optional)** | Summaries & reports                             |
